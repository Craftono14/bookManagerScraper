# Book Search Discord Bot

- Place your Discord bot token in `config.json` (copy `config.json.example`).
- Install dependencies:

```bash
npm install
```

- Run the bot:

```bash
npm start
```

- Usage in Discord channel (bot needs Message Content intent enabled):
  - `!search seven seas` (defaults to `keyword` search type)
  - `!search seven seas publisher` (explicit `publisher` type)

- Results: bot will reply with a CSV file containing columns: `Book Name,Book Price,Store URL,Item Url`.

Notes:
- `urlList.csv` contains store base URLs; `aiInfo.txt` shows the URL template used.
